# 📝 Blog Web App

A modern full-stack blogging platform built with a powerful and scalable tech stack, focusing on performance, type safety, and clean UI.

---

## 🚀 Tech Stack

### 🔧 Backend

* **Runtime**: Cloudflare Workers (Edge-based deployment)
* **Framework**: Hono (lightweight & fast)
* **Language**: TypeScript
* **Validation & Auth**: Zod
* **Database ORM**: Prisma

### 🎨 Frontend

* **Library**: React (TypeScript)
* **Styling**: Tailwind CSS

---

## ✨ Features

* 🔐 Secure authentication with validation
* ⚡ Fast API using edge functions (Cloudflare Workers)
* 📝 Create, read, and manage blog posts
* 🎯 Type-safe full-stack development
* 💅 Clean and responsive UI

---
